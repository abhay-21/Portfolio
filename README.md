## Getting Started

### Prerequisites

No prerequisites needed! This is a static website that runs entirely in the browser.

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! The website should load immediately.

### Local Development

If you want to make changes:

1. Open the project folder in your code editor
2. Edit the HTML, CSS, or JavaScript files as needed
3. Refresh your browser to see changes

## Customization

### Update Personal Information

Edit `index.html` to update:
- Contact information (email, phone, social links)
- About section text
- Project descriptions
- Education details

### Change Colors

Edit `styles.css` and modify the CSS variables in the `:root` selector:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... other variables */
}
```

### Add New Sections

1. Add HTML structure in `index.html`
2. Add corresponding styles in `styles.css`
3. Update navigation menu if needed

## Deployment & Live Link

### Quick Deployment (Recommended: GitHub Pages)

1. **Create GitHub Repository:**
   - Go to https://github.com and create a new repository
   - Name it: `portfolio` or `abhay-jaiswal-portfolio`
   - Make it **Public**

2. **Upload Files:**
   - Upload all files (index.html, styles.css, script.js) to the repository

3. **Enable GitHub Pages:**
   - Go to Repository Settings ‚Üí Pages
   - Select "main" branch as source
   - Your portfolio will be live at: `https://yourusername.github.io/portfolio/`

4. **Alternative: Netlify/Vercel**
   - Drag and drop your portfolio folder to https://netlify.com or https://vercel.com
   - Get instant live URL!
